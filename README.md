# bitscube_daisyling

## 需求
 1. 首页：搜索、音乐列表、视频列表（下拉分页）
 2. 搜索：搜索历史、搜索结果（下拉分页）
 3. 音乐/视频列表：下载、暂停、播放、收藏
 4. 我的：我的下载、我的收藏、我的足迹、我的播放、清除缓存、版本信息、退出登录等
